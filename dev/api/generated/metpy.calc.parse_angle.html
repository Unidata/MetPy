parse_angle¶

metpy.calc.parse_angle(input_dir)[source]¶

Calculate the meteorological angle from directional text.

Works for abbrieviations or whole words (E -> 90 | South -> 180) and also is able to parse 22.5 degreee angles such as ESE/East South East

Parameters

input_dir (string or array-like) – Directional text such as west, [south-west, ne], etc

Returns

pint.Quantity – The angle in degrees
