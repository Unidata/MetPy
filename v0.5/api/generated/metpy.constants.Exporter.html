Exporter¶

class metpy.constants.Exporter(globls)[source]¶

Manages exporting of symbols from the module.

Grabs a reference to globals() for a module and provides a decorator to add functions and classes to __all__ rather than requiring a separately maintained list. Also provides a context manager to do this for instances by adding all instances added within a block to __all__.
